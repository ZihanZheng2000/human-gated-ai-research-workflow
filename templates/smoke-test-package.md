# Smoke-Test Package

This compact package is a lightweight diagnostic index. It is not a replacement for the full Planning, Modeling, Reporting, and Reviewing packages when the project is being developed as a real research artifact. Use it to confirm that the workflow can run end to end, then keep the formal stage packages as the authoritative record for substantive work.

Use this compact package when the goal is to test the workflow itself rather than produce a publication-ready manuscript. A smoke test should still run Planning -> Modeling -> Reporting -> Reviewing. It may collapse the stage outputs into this shorter artifact when the project is intentionally small.
